WHAT I BUILT

Rift Record 구현 내용

Riot API Integration

Riot ID, 최근 솔로 랭크, 랭크 정보와 챔피언 숙련도를 서버 API로 조회합니다.

Match Data Analysis

KDA, CS/분, 킬 관여율, 피해량과 시야 점수를 최근 15경기 기준으로 분석합니다.

Custom Champion Tier

수집된 participant 데이터로 라인별 자체 티어를 계산하고 Low Sample을 표시합니다.

Secure API Handling

Riot API Key와 Supabase Service Role Key를 서버 환경변수로만 관리합니다.

Persistent Database

Supabase에 match와 participant를 저장해 Vercel에서도 수집 결과를 유지합니다.

Demo Mode

API Key나 DB 설정과 무관하게 주요 분석 기능을 확인할 수 있는 mock data를 제공합니다.
